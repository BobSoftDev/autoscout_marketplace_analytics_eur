# Core Calculated Fields

Total Revenue (EUR)
SUM([revenue_eur])

Revenue per Lead (EUR)
SUM([revenue_eur]) / SUM([contacts])

Revenue per Listing (EUR)
SUM([revenue_eur]) / SUM([active_listings])

Contact → Sale Rate
SUM([sales]) / SUM([contacts])

Search → Contact Rate
SUM([contacts]) / SUM([searches])
